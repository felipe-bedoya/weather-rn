# Reservamos Weather App

# Prerequisites

Make sure you have at least:
- Node.JS v18.19.0
- React Native 0.73.4
- Expo 50.0.6

# Instalation
- Clone the repository
- Install dependencies `npm install`
- Run with `expo start`

Or you can directly run an emulator or simulator with
`expo start --ios` or `expo start --android`
